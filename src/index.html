Help

Model

Garis

1. Klik tombol "line" untuk memulai menggambar garis
2. Klik pada canvas untuk menentukan titik pertama
3. Klik kembali pada canvas untuk menentukan titik kedua

Persegi

1. Klik tombol "square" untuk memulai menggambar persegi
2. Klik pada canvas untuk mengambil titik awal
3. Klik kembali pada canvas untuk mengambil titik diagonal

Persegi Panjang

1. Klik tombol "rectangle" untuk memulai menggambar persegi panjang
2. Klik pada canvas untuk mengambil titik awal
3. Klik kembali pada canvas untuk mengambil titik diagonal

Polygon

1. Klik tombol "polygon"
2. Klik pada canvas untuk mulai mengambil titik
3. Ulangi langkah kedua hingga telah selesai menentukan titik
4. Klik kanan untuk menyelesaikan gambar

Polygon Convex Hull

1. Klik tombol "polygon Convex Hull"
2. Klik pada canvas untuk mulai mengambil titik
3. Ulangi langkah kedua hingga telah selesai menentukan titik, titik bebas untuk ditekan dimana saja
4. Klik kanan untuk menyelesaikan gambar

Fitur

Edit

1. Pilih objek yang ingin diedit pada select object di kanan atas
2. Klik tombol select
3. Pilih fitur yang ingin diedit

  • Color : Pilih warna yang diinginkan
  • Translasi X : Gerakkan slider untuk menggeser objek di arah sumbu x
  • Translasi Y : Gerakkan slider untuk menggeser objek di arah sumbu y
  • Rotasi : Gerakkan slider untuk memutar objek
  • Dilatasi : Gerakkan slider untuk memperbesar objek
  • Shear : Gerakkan slider untuk menggeser objek ke arah sumbu x
  • Size :
    1. pilih metode yang ada pada pilihan metode spesial (optional)
    2. klik titik yang akan diedit pada canvas
    3. ubah sesuai kemauan
    4. klik pada canvas untuk menyelesaikan edit

    • Save

    1. Tulis nama file pada kolom input, pilih save object untuk menyimpan beberapa object saja dan pilih save canvas untuk menyimpan semua object. Memilih object dilakukan melalui select object di kanan atas
    2. Klik tombol save

    Load

    1. Klik tombol load
    2. Pilih file yang ingin dimuat
    3. Klik open
